# blog
## 项目结构目录
+ db 数据存储目录
+ models 数据库模型文件目录
+ public 公共文件目录（js,css,img等资源）
+ routers 路由配置文件目录
+ schemas 数据库结构文件（schema）目录
+ views 模板视图文件目录

## 技术框架
+ nodejs
+ express nodejs搭建应用的框架
+ bodyParser 解析post请求数据
+ cookies 读写cookie
+ swig 模板解析引擎
+ mongoose 操作mongoodb数据
+ markdown markdown语法解析生成模块
